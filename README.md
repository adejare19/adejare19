# Evergreen Academy Website Template

A production-ready, multi-page private school website template built with semantic HTML5, modular CSS3, and vanilla JavaScript.

## Pages Included

- `index.html` — Home page with hero CTA, facilities, testimonials, and admissions call-to-action
- `about.html` — Mission, vision, values, principal message, and gallery
- `admissions.html` — Step-by-step admissions process, fee table, and inquiry form
- `contact.html` — Contact form, business hours, address, social links, and map placeholder

## Tech Stack

- HTML5 (semantic structure)
- CSS3 (custom design system, responsive Grid/Flex layouts)
- Vanilla JavaScript (mobile menu + form validation)

## Features

- Sticky, accessible navigation with mobile toggle
- Reusable component classes and consistent spacing tokens
- Basic frontend form validation with inline feedback
- SEO essentials (`title`, `meta description`, Open Graph tags)
- Accessibility essentials (skip link, heading hierarchy, ARIA labels, accessible forms)

## Local Preview

Open any HTML file directly in a browser, or run a simple static server:

```bash
python3 -m http.server 4173
```

Then visit:

```text
http://localhost:4173/index.html
```

## Customization Notes

To adapt for a real school:

1. Replace school name, logo/brand initials, and contact details.
2. Update social profile URLs and admissions/fee data.
3. Replace stock image URLs with licensed school-specific assets.
4. Connect forms to a backend endpoint or form service.
